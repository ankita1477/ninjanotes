## 🚨 Troubleshooting

<details>
<summary>Common Issues</summary>

1. **FFmpeg Not Found**
   ```cmd
   setx PATH "%PATH%;C:\ffmpeg-7.1-essentials_build\bin"
   ```

2. **Port 5000 Busy**
   ```python
   app.run(port=5001)
   ```

3. **Memory Error**
   - Close other applications
   - Free up RAM
   - Check file size (<16MB)

</details>
